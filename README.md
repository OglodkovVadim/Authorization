# Authorization example

This is an example of simple authorization for a GUI application written in C++. Working with [SQL](https://ru.wikipedia.org/wiki/SQL) at a basic level is presented here.
___

###  Frontend stack
+ Qt Design (Widgets)
___
### Backend stack
  + PostgreSQL 
  + Qt (Qt Creator)
